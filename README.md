# Cubid Flutter Plugin

## 기본 요건
* Flutter SDK 3.13.1 이상
* 권장사항: Flutter SDK 3.27.1 이상

### Android: 최신 버전의 Android Studio
* Gradle 7.0.1 이상
* minSdkVersion 21 이상
* OS 지원 Android 8.0 이상

### iOS: 최신 버전의 Xcode와 명령어 도구 활성화
